优秀文章

[原创] 每周 Swift 社区问答 2016-01-06 (@shanks)  

翻译组的原创文章，每周整理 Stack Overflow 和 Swift 官方论坛中的提问及回答，推荐阅读。

[译] 聚焦 Swift #2 (@pmst)  

一个 commit 修复编译器 91% 的报错？来看看 Swift 源码最近又有什么新动态。

[译] 在 Target-Action 中使用响应链 (@小铁匠Linus)  

响应链是 iOS 中非常重要的一个机制，本文会介绍什么是响应链以及如何在 Target-Action 中使用它。

[译] 如何使用 iOS 9 App 瘦身功能 (@JackAlan)  

App Thinning 是 Xcode 的新功能，开启之后用户可以只下载自己需要的应用文件和资源，从而加快下载速度，减少内存占用。

[译] iOS 并发：从 NSOperation 和 Dispatch Queues 开始 (@ray16897188)  

长篇教程，详细介绍 iOS 并发的相关知识，存下来慢慢读吧。

Swift 2.2 Release Process  

本周英文文章，苹果官方的 Swift 2.2 发布进度，从 2016.1.13 开始 Swift 2.2 会在单独的分支进行开发，master 分支用来开发 Swift 3。

iOS & Mac OS 开源项目

pomo.tv：在线观看技术分享 (@Chris Eidhof)  

Chris Eidhof 大神开发的网站，目前已经有超过 500 个视频，专注于 Mac、iOS 和 Swift 开发。

bitbar：在菜单栏上显示任何内容 (@matryer)  

Mac 专用，可以把脚本和程序的输出显示在屏幕右上角的菜单栏中，做一些自己的小工具会方便很多。

跨平台开源项目

Swift socket 编程 (@iachievedit)  

学习在 Linux 上用 Swift 操作 socket，写一个自己的服务器吧。

UnchainedString：扩展 Swift 字符串功能 (@dunkelstern)  

Linux 版本的 Swift 缺少很多重要的字符串处理函数，这个库提供了字符串查找和替换功能。

Working with very large numbers in Swift - BigInt (@beaunouvelle)  

本周的英文文章，介绍如何使用字符串来实现超大整数的操作。

开源项目

MissionControl-iOS：远程配置库 (@Marko)  

应用大了就会有各种功能，在目前苹果的审核机制下要想实现实时改动功能就必须从服务器端下发配置。这个库封装了各种远程配置常见的操作，可以在获取配置之后通知变动的配置项，还配有一个非常可爱的火箭发射 Demo。

Stellar：物理动画库 (@August)  

简化 UIKit Dynamics 的操作，轻松实现各种动画效果。

Clipy：OS X 剪切板管理库 (@Clipy)  

这个项目最大的亮点是 Swift OC 混编，包含各种复杂的剪切板操作，可以学习一下如何优雅地混编代码。

JSONExport：JSON 对象导出工具 (@Ahmed-Ali)  

OS X 的应用，可以把 JSON 导出成不同语言中的数据结构和类，比如 Java、Swift 中的类、Swift 中的结构体，可以用来转换 API 数据，转换完毕直接使用。

Scrollable-GraphView：炫酷图表 (@philackm)  

图表视图，自带动画，支持滚动，非常适合展示一系列数据点。

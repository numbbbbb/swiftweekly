优秀文章

[原创] 每周 Swift 社区问答：@NSApplicationMain 和 @UIApplicationMain (@shanks)  

翻译组的原创文章，每周一个主题，收录相关的经典问题和解决方法，推荐阅读。

[译] 准备好迎接 3.0 API 变化 (@zltunes)  

Swift 3.0 API 命名方式会大改，作者总结了几条改动规则，帮助你更好地“重新入门 Swift”。

[译] GCD 使用指南 (@walkingway)  

深入浅出 GCD，看完就能直接实践，GCD 没你想得那么难。

[译] 度量单位 (@Prayer)  

代码中出现大量数字的时候如何明确数字的含义？是时间、金额还是其他？来看看作者如何把数字的含义巧妙地加入代码。

去model化和数据对象 (@Casa Taloyum)  

项目中有 model 层是非常常见的设计，但是引入 model 会给代码引入新的依赖，有时我们并不想要这样的副作用。如何去 model？作者给出了几种方式。

Scripting in Swift (@Sam)  

作为一个一统宇宙的语言，写脚本显然是必备特性。本文教你如何写一个 Swift 脚本，从多个 markdown 文件读入数字，渲染成 HTML，并输出到一个文件中。

开源项目

StateView：根据数据自动更新 View (@Sahand)  

有点 MVVM 中 binding 的意思，也有点 React 中 diff 算法的意思。数据改动之后计算需要变更的 UI 元素，然后改动 View，很有意思。

Screenotate：自动标注你的屏幕截图 (@Omar)  

Mac 的截图文件名中只有日期时间，要找东西很不方便。这个库可以检测截图时所处的应用或者网页，把一些重要的信息加入到文件名中，帮你快速找到想要的截图。

GRDB.swift：多功能 SQLite 工具库 (@Gwendal)  

最大的亮点是：可以像操作普通对象一样操作数据库中的行。

Interpolate：给手势加上动画 (@Roy)  

给各种手势加上默认的动画效果，当然，你也可以自定义你想要的动画。

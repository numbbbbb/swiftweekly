优秀文章

[译] 混淆 API 密钥 (@pucca601)  

API 用起来很方便，但是密钥的安全性是个大问题。本文提到了一种密钥的加密和解密方法，供大家参考。

[译] 《重构与模式》Swift 版之参数对象 (@Channe)  

这次来重构参数对象。如果参数有多个，或者需要经常变动，可以构建一个参数对象，从而减少代码改动。文章浅显易懂，推荐阅读。

[译] 使用 Xcode 8 为 iMessage 创建表情包插件 (@小袋子)  

新发布的 iMessage 增加了许多扩展功能，其中的一大亮点就是表情包。表情包插件的开发非常简单，如果你有兴趣可以来尝试一下。

在实践中应用 RxSwift (@DianQK)  

今天是一篇原创文章，作者是翻译组的@靛青K 。他最近在写一系列 RxSwift 文章，大家可以帮忙提提建议：http://book.dianqk.org。

Pokemon Go 锁区破解 (@txx)  

虾神的新文章，几行代码就可以破解锁区。当然这里的重点不在游戏，而是破解方法，推荐跟着步骤操作一遍，会对 iOS 应用有更深入的了解。

Xcode 8: How To Create an Animated iMessage Sticker (@Natasha)  

本周的英文文章，介绍如何创建动态 iMessage 表情，是时候做一套来炫耀了。

开源项目

BarcodeScanner：条码扫描库 (@hyperoslo)  

带状态控制的条码扫描库，支持处理相机权限、自定义颜色和提示信息，不依赖其他第三方库。

Potatso：Shadowsocks iOS 客户端 (@icodesign)  

基于 iOS 9 的 NetworkExtension 框架实现 Shadowsocks 代理，由国人开发，虽然还有很多问题不过确实值得期待。

SideMenuController：侧边栏控制器 (@teodorpatras)  

支持自定义位置、配置动画效果和自定义状态栏行为。

KYShutterButton：模拟 iOS 相机按钮 (@ykyouhei)  

非常有趣的一个项目，模拟了 iOS 相机按钮，包括慢动作和延时效果，可以自定义颜色。

SwiftLuhn：验证信用卡号合法性 (@MaxKramer)  

信用卡号可以用 Luhn 算法验证，除了验证卡号合法性，还可以判断卡片类型，能够识别包括 Visa 在内的六种卡片。

UPCarouselFlowLayout：轻松实现走马灯效果 (@ink-spot)  

走马灯效果和图片轮播类似，区别是当前显示的内容会被放大，手指滑动就可以左右切换内容，用好了确实比较炫酷。

TinyPNG4Mac：Mac PNG 压缩工具 (@kyleduo)  

TinyPNG 是一个很出名的 PNG 压缩网站，作者给它开发了一个 Mac 客户端，直接拖拽就可以压缩 PNG 图片。

XEasyMotion：用键盘控制鼠标 (@h2ero)  

非常有意思的一个项目，可以用键盘实现区域缩放、移动、点击、右击等等鼠标操作，配合脚本可以做一些有趣的事。

reminders-cli：用命令行操作提醒事项 (@keith)  

用命令行操作 Mac 上的提醒事项，可以查看、添加、完成项目，这里的重点是如何用 Swift 编写一个命令行工具。

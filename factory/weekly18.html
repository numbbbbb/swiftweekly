[译] 如何成为一名入门级 iOS 开发者 (@ray16897188)  

成为入门级 iOS 开发者需要具备什么技能？需要学习什么工具？来看看这位作者总结的内容。

iOS & Mac OS 开源项目

Material：Material Design 风格控件 (@CosmicMind)  

Material Design 是 Google 推出的风格指南，现在你也可以在 iOS 中使用了。

FileBrowser：手机上的 Finder (@marmelroy)  

按照 Finder 风格制作的 iPhone 文件管理器，支持 3D Touch，支持预览多种文件，学习源码的好项目。

SPTDataLoader：Spotify 官方 HTTP 库 (@spotify)  

Spotify 是一款评价极高的音乐软件，最近开源了他们 iOS 端的 HTTP 库，来看看 Spotify 的代码是否像他们的设计一样牛。

跨平台开源项目

blackfish (@elliottminns)  

受 Node.js 和 Express 启发，作者用 Swift 开发了一个 web 框架，写过 Express 的同学迁移过来会很轻松。

PMJSON (@postmates)  

纯 Swift JSON 解析器，可以在 Linux 中使用。虽然项目很前卫，但是命名仍然是一股浓浓的 OC 风……
